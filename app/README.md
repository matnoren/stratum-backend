# Stratum Backend

Backend service built with Python and FastAPI for managing users, authentication and internal processes.

## Features
- REST API
- JWT Authentication
- User and role management
- PostgreSQL integration
- Environment-based configuration
- Ready for cloud deployment (AWS)

## Tech Stack
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- Docker

## Running locally
1. Create a virtual environment
2. Install dependencies
3. Set environment variables
4. Run the application
